# Evidence Requirements v1

## Refinement Exit (Ready for Delivery candidate)

- Stories linked to Epic
- Measurable Acceptance Criteria per story
- NFR checklist completed or marked N/A
- Open questions resolved or explicitly tracked
- Solution Design v1 link
- ADR links for key decisions
- Tasks/subtasks linked
- Security review status recorded

## Delivery Exit (Ready for Release candidate)

- CI test results link
- SAST results link
- Dependency scan link
- Secrets scan link
- IaC scan link
- QA e2e/regression evidence link
- Security sign-off link if Risk is Medium/High

## Release Exit (Done candidate)

- Release notes link
- Runbook link (with rollback)
- Deployment evidence link

## Approval Requirements

- Budget exception requires linked approval artifact
- Policy gate change requires linked approval artifact
